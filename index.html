Apex-tools

Satu perangkat tunneling untuk mendapatkan akses kebebasan dari kungkungan provider mobile di Indonesia

Download Download View Apex-tools on GitHub

Merdeka!!!.

Mari berbagi sepenggal waktu diantara kesibukan anda untuk membuat satu tools injekan yang benar-benar handal dan memberi kemudahan kepada penggunanya untuk berinternet secara gratis tanpa membebani pihak Operator Seluler (Opsel).

  • Sebagai media awal menuju ke arah sana kita buat tools ini memnggunakan bahasa pemrograman Visual Basic
  • Kumpulan bugs diperoleh dari aneka tools dan pemberian secara suka rela dari berbagai pihak

Untuk komparasi file dan download langsung ke desktop anda bila anda pengguna linux atau Windows XP/7/Vista pergunakan aplikasi SSH tools seperti prutty, wcp, dll dengan command sbb:

$ cd folder_dimana_anda_buat/apex-tools $ git fetch origin $ git checkout gh-pages


Bila anda pengguna Mac, simply sync your repository and you'll see the new branch.

### Designer Templates
Tools ini dirancang untuk bisa berjalan secara polosan dan mendukung SSH tunnel baik menggunakan bitvise+proxifier maupun aplikasi tunneling lainnya. Dengan demikian pada penampilannya diberikan opsional penggunaannya.

### Mau menyimpan di Flasdisk USB?
Bila anda tidak menghendaki otomasi file yang ada di sini silahkan tekan tombol a branch named `gh-pages` ke repository anda dengan nama halaman lain.
